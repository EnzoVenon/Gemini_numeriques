## Installation: 
```
git clone https://github.com/EnzoVenon/Gemini_numeriques.git
```
- install node (and npm)
- go to the folder:  Gemini_numeriques an open it in ther terminal 
- in the terminal, type these commands

```bash
npm install
npm run build
npm run start
```
